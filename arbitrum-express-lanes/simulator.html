Block Simulator

Mine two linked blocks in your browser using the real Web Crypto SHA-256 implementation. Change Block 1's data after mining and watch Block 2 break — that's chain immutability in action.

Mine Block 1 first — Block 2 needs its hash.
How "mining" works here: the button repeatedly increments the nonce and re-hashes previousHash + data + nonce with SHA-256 until the output starts with 00 (a simplified stand-in for Bitcoin/Ethereum's real proof-of-work target). Block 2 stores Block 1's hash as its own "previous hash" field — so editing Block 1's data after the fact changes Block 1's hash, which no longer matches what Block 2 has on record, and Block 2 flips to Invalid.
